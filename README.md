Для запуска приложения необходимо:
1) проверить текущую конверсию CRLF в LF командой **git config core.autocrlf**
2) отключить конверсию **git config --global core.autocrlf** false для корректной отработки .sh скрипта docker-контейнера
3) в корневой папке прописать **docker-compose up**. При запуске автоматически запустятся тесты, информация о которых будет выведена в терминал
4) swagger будет доступен по урлу http://127.0.0.1:8000/schema/swagger
5) после завершения работы можно вернуть **git config core.autocrlf** в прежнее значение
