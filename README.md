Для запуска приложения необходимо в корневой папке прописать **docker-compose up**. При запуске автоматически запустятся тесты, информация о которых будет выведена в терминал
Если при запуске контейнер с app будет падать в ошибку "exec /entrypoint.sh: no such file or directory", то надо поменять line-separator с CRLF на LF внутри этого файла
